Layanan Barimetri

Layanan survei barimetri untuk mengukur kedalaman dan kontur dasar perairan secara akurat, mendukung perencanaan konstruksi pelabuhan, bendungan, hingga proyek reklamasi.

Deskripsi Layanan

Survei barimetri dilakukan untuk mengetahui profil dasar perairan dengan mengukur kedalaman menggunakan alat sonar atau echosounder. Hasilnya digunakan untuk pemetaan kontur dasar laut, sungai, atau danau.

Data ini membantu proses perencanaan teknis seperti pembangunan dermaga, pengerukan, dan pengendalian sedimentasi.

Manfaat Survei Barimetri

  • Menyediakan peta kontur dasar perairan
  • Menentukan kedalaman dan morfologi dasar air
  • Mendukung desain pelabuhan, bendungan, dan reklamasi
  • Memberikan data presisi untuk perencanaan hidroteknik

Peralatan yang Digunakan

Echosounder / Sonar

Alat utama untuk mengukur kedalaman perairan dengan gelombang suara.

GPS & Perangkat Navigasi

Menentukan posisi horizontal agar hasil pengukuran terpetakan dengan tepat.

Software Pengolah Data

Digunakan untuk memproses dan menampilkan hasil survei dalam bentuk peta kontur digital.
